# Personal Expense Tracker (React)

This workspace now contains a React-based personal expense tracker application.

The original vanilla JavaScript version is referenced in `expense-tracker-README.md` but the main project lives in `react-expense-tracker`.

### Getting started

```bash
cd react-expense-tracker
npm install    # install dependencies
npm start      # run development server
```

Build for production:

```bash
npm run build
```
